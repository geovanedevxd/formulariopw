    2. Placeholder E Utilizado para indicar ao navegador o texto que deverá aparecer escrito no campo do formulario
  2. Na tag <input>> com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
    3. O Atributo Checked e usado para indicar quais checkbox e qual radio button devem estar ligados inicialmente se necessário
  3. Na tag <input> para que serve o atributo "value"?
    4. O Atributo value para definir o valor a ser enviado por este item.

  5. Para que serve a tag
    6. A tag "label" em HTML e usada para criar rotulos de texto associados a elementos de formularios, como caixa de seleção, bottoes de opção e campos.
  6. Na <tag> para que serve o atributo "selected"?
    7. O atributo "selectd" na tag option em html e usada para indicar qual opção de seleção para ser predefinida ou pre-selecionada quando a página carrega.
  7. Nas tags de formulário para que serve o atributo "required"?
    8. E um elemento usado para indicar que o campo e Obrigatorio preencher antes de o formulario pode ser enviado.
  8. Para que serve a tag <textarea>?
    9. A tag é usada para criar um campo de entrada de texto multilinha em um formulário web.
  9. Na tag <form> para que serve o atributo "action"?
    10. O atributo action define o local (através de uma URL) ao qual serão enviados todos os dados recolhidos do formulário
  10. Na tag <form> para que serve o atributo "method"?
    11. Essa tag indica que estamos iniciando um formulario, recebe como principais atributos method que recebe como valor o método http que esse formulário ira executar (get,post) e action que especifica para onde enviar os dados do formuário quando um formulário é enviado.
  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag?
    12. Permite que o usuário digite um valor que não é exibido de forma legível, sendo utilizado para a entrada de senhas.Nesse caso, o elemento <form> deve ter o atributo method definido com o valor post, para que a senha nao seja mostrada na linha de endereço.
  12. Qual valor deve ser informado no atributo "method" na tag <form> em um formulário que será usado para fazer upload de arquivos?
    13. O atributo action define o local (Uma URL) em que os dadod recolhidos do formulário devem ser enviados
